Hi, my name is Mariana and I have some advice for you!



Code everyday!

Even if all you can do is five minutes a day - do it! Practice, practice and more practice doesn't make you perfect, it gives you the confidence to tackle anything that will come up and something always comes up!.

If you don't have the time to sit down at your computer, you can download apps to your phone and code while on the move. I use Solo Learn and they have everything from HTML to PHP and much more.

Google and the interweb is your friend

Whenever you run into a problem or you forget a bit of code, ask Google! It's a guarantee that someone else has encountered the same problem you have - don't reinvent the wheel!

You don't have to learn everything

HTML, CSS, Java, C++, Javascript, JQuery, PHP, Ruby, Python, React, Angular...and more! There are a lot of programs out there, but you don't have to learn everything. You'll eventually start to prefer some over others - focus on those.

It's OK to take a break

It's important to step away from the screen sometimes! Go outside, talk to humans, eat some chocolate - the code will still be there when you get back.
