Hi my name is Rebecca and I have some advice for you!

Don't Give Up!

It may seem daunting at times. Stick with it!

Try to spend a small amount of time everyday learning.

It's similar to learning music.

When learning the piano, one must first learn basic theory. This is difficult, boaring, and often discouraging. But once you understand note values and names, you can start creating music!

The same is true for coding. Have patience in the begining. Once you've learned the basic "theory" you'll be able to create beautiful websites!
